Samples from the Missioneer project.


Here is a synopsis of the site's development.

Setup a basic Wordpress Installation.
Started with the standard TwentyFifteen template.
Created the basic pages.

Used Photoshops to name the layers well and put them in correct hierarchy for importing into Dreamweaver.

Used Photoshop's Extract Assets and Slicing tools to export images assets needed for the design as well as used Dreamweaver's Extract feature to import assets.

Used bootstrap and image assets to create 
a child theme from TwentyFifteen.

Used jQuery to add functionality to videos and
sliding transitions.

Ensured responsive design principles while
working with designer to create proper user
experience.
